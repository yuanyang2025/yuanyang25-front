# yuanyang25-front

`main` branch automatically deployed on [https://2025.yuanyang.app](https://2025.yuanyang.app). Always use pull request and **never** push directly into `main`!

# Scripts:

Run locally:

```bash
npm run dev
```

Test buildability locally:

```bash
npm run build
```

Formatting before push:

```bash
npx prettier --write .
```
